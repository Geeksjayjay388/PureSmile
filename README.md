## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS version recommended)
- `npm` or `yarn`

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd puresmile
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
